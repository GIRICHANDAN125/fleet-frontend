# DECISIONS.md

## Decision 1: React for Frontend

Options:

* React
* Angular
* Vue

Chosen:

* React

Reason:

* Component reusability and strong ecosystem.

## Decision 2: Node.js Backend

Options:

* Node.js
* Django

Chosen:

* Node.js

Reason:

* JavaScript across full stack.

## Decision 3: MongoDB Database

Options:

* MongoDB
* PostgreSQL

Chosen:

* MongoDB

Reason:

* Flexible document-based schema.

## Decision 4: JWT Authentication

Options:

* Session Authentication
* JWT

Chosen:

* JWT

Reason:

* Stateless authentication and easier API integration.
